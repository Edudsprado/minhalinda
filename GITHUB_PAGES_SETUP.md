## 🔗 Domínio Customizado (Opcional)

Se desejar usar um domínio customizado gratuito:

### Opção 1: Freenom (Domínios .tk, .ml, .ga)
1. Acesse: https://www.freenom.com
2. Registre um domínio gratuito (ex: `love.tk`)
3. Em "Management Tools" > "Nameservers"
4. Use os nameservers do GitHub Pages (veja em Settings > Pages)

### Opção 2: Usar CNAME no Repositório
1. Crie arquivo `CNAME` na raiz do repositório com seu domínio:
   ```
   love.tk
   ```
2. Push e configure os nameservers
