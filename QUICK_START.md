## ❓ Dúvidas?

### "Não consigo fazer commit"
- Certifique-se que tem Git instalado: `git --version`
- Verifique se está na pasta correta
- Abra um novo terminal (PowerShell ou CMD)

### "GitHub Pages não está ativando"
- Verifique se o repositório é público
- Tente fazer refresh na página Settings
- Aguarde mais tempo (pode levar 5 minutos)

### "Não vejo as mudanças no celular"
- Limpe o cache (Ctrl+Shift+Delete ou Cmd+Shift+Delete)
- Tente em modo incógnito
- Aguarde o GitHub processar (2 min)

### "Galeria não está em 2 colunas no celular"
- Verifique o tamanho: deve estar < 480px
- Use DevTools para confirmar media query ativa
- Tente em outro navegador
